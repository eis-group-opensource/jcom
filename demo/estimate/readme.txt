サンプル１

●想定事例:Excelを使った見積書の自動生成

ＪａｖａからＥｘｃｅｌで作られたシートに値を格納し、
印刷を行います。また、それをファイルに保存します。
具体的に、ある見積書を自動的に作るシステムがあると仮定します。
まず、"estimate.xls"を見てください。 このファイルをいったん、
別の名前でコピーし、開き、必要な箇所に値を入れていきます。
そして印刷し、保存して終了します。

コピーされたファイルは、すべて保存するものとします。
従って、ファイル名はユニークな値にします。これは、
見積書Noと一致させる方がよいでしょう。 それらは、
見積有効期間が過ぎるまで保持します。
ただし、このサンプルではそこまで実装していません。

サンプルでは実際には印刷しません。しかし、以下の行を
コメントから外せば印刷を行います。
//     est.print_enable = true;

実際の運用となると、印刷の代わりにＦＡＸやメールに送ることも
考えられますね。

●実行方法

あらかじめ、ＪＣｏｍが正しくインストールしているものとします。
インストール方法は docs/index.html の中の「パッケージ jp.ne.so_net.ga2.no_ji.jcom 」
dの説明を参照して下さい。
コンパイル方法          > javac EstimateMaker.java
実行方法                > java EstimateMaker


註：FileCopy.java は O'REILLY "JAVAプログラミング クイックリファレンス"
の"Java Exampleオンライン" http://www.oreilly.com/catalog/books/jenut/
からダウンロードしました。
